# PAMAgents

PAMAgents is a CCTV monitoring project that combines a Go backend, a React frontend, and an AI engine to support workplace safety monitoring. The project is used to process video streams, display a web interface, and run PPE detection such as helmets and safety vests.

## Requirements

- Go
- Node.js and npm
- Python 3.10+
- Python dependencies listed in `backend/engine/requirements.txt`

## Notes

Engine configuration and detection policy files are available in `backend/engine/` and `backend/policy/`.

## Summary

This project is designed to make CCTV-based safety monitoring easier with a web interface and automated video analysis.
